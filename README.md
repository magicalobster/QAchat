# QAchat
QA Chat is an AI quiz application platform based on Spring Boot, Redis, ChatGLM, RxJava, and SSE. Through this platform, users can quickly generate questions and create quiz applications using LLM. Once approved, it enables online quiz-taking and provides answer summaries based on various scoring algorithms or AI analysis.
